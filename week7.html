📤 Output Devices – Overview

  • Output devices send processed data from a computer to the user.
  • They present data in the form of text, images, audio, or video.
  • They help users understand and use information generated by a computer.

🗂 Types of Output Devices

  • Visual: Monitors, projectors, VR headsets, AR glasses
  • Printers: 2D printers, 3D printers, plotters, copy machines
  • Audio: Speakers, headphones, sound cards
  • Tactile: Braille displays, haptic devices
  • Specialized: Medical imaging, brain-computer interface devices

🖥 Monitors

  • Also known as Visual Display Unit (VDU).
  • Displays output using pixels.
  • Image sharpness depends on number of pixels.

📺 Cathode Ray Tube (CRT) Monitor

  • A CRT monitor is an output device that uses a vacuum tube.
  • Images are produced by firing electron beams onto a phosphorescent screen.
  • The screen is made of very small dots called pixels.
  • Smaller pixels result in better resolution and sharper images.
  • Multiple illuminated pixels combine to form text and graphics.
  • The screen is divided into fixed character boxes.
  • Only a limited number of characters can be displayed at one time.
  • Standard CRT display shows 80 characters per line and 25 lines per screen.

🧩 Components of CRT Monitor

  • Electron Gun: Generates a beam of electrons.
  • Phosphor Screen: Emits light when struck by electrons.
  • Deflection Yoke: Controls direction of the electron beam.
  • Circuit Board: Manages internal control and connectivity.

⚖ Advantages and Disadvantages of CRT

Advantages

  • High refresh rate
  • No motion blur
  • Good color accuracy
  • Supports multiple resolutions

Disadvantages

  • Bulky and heavy
  • Consumes more power
  • Produces heat
  • Requires more space

🖨 Printers

  • Produce hard copies of digital data.
  • Impact Printers: Use ribbon and physical contact.
  • Non-Impact Printers: Faster, quieter, better quality.

🦠 Computer Virus

  • A computer virus is a type of malicious software.
  • It can replicate itself by inserting its code into other programs.
  • The virus executes when the infected program is run.
  • Viruses spread through files, emails, and removable media.

⚠ Effects of Computer Virus

  • Destroys or corrupts files and data
  • Slows down system performance
  • Consumes memory and CPU resources
  • Logs keystrokes and steals information
  • Causes system crashes
  • Can completely wipe hard drives

🧪 Types of Computer Virus

  • Boot Sector Virus: Attacks system startup files.
  • File Virus: Infects executable files.
  • Email Virus: Spreads via email attachments.
  • Macro Virus: Infects documents using macros.
  • Polymorphic Virus: Changes form to avoid detection.
  • Multipartite Virus: Attacks boot sector and files.
  • Encrypted Virus: Uses encryption to hide code.
  • Stealth Virus: Hides its presence.
  • Resident Virus: Remains active in system memory.
  • Direct Action Virus: Executes only when file runs.
  • Browser Hijacker: Alters browser settings.
