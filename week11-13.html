📡 Data Communication

  • Data communication is the exchange of data between devices.
  • It requires sender, receiver, medium, message, and protocol.

📻 Analog Transmission

  • Uses continuous signals.
  • Examples: radio waves, sound waves.
  • Uses amplitude and frequency.

💻 Digital Transmission

  • Uses discrete signals (0 and 1).
  • More reliable and less noise-sensitive.

🧱 OSI Model

  • 7-layer communication model.
  • Standard framework by ISO.

🪜 OSI Layers

LayerFunction
ApplicationUser interface
PresentationEncryption, compression
SessionSession management
TransportFlow & error control
NetworkRouting
Data LinkFraming
PhysicalBit transmission

📜 Protocol

  • A protocol is a set of rules that governs data communication.
  • It defines how data is transmitted between devices.
  • Ensures proper understanding between sender and receiver.
  • Main elements of protocol:
    • Syntax: Structure and format of data.
    • Semantics: Meaning of each section of data.
    • Timing: Speed of transmission and synchronization.

🌐 TCP/IP Model

  • TCP/IP stands for Transmission Control Protocol / Internet Protocol.
  • It is the practical model used in real networks.
  • Developed by ARPANET.
  • Consists of four layers.

🪜 TCP/IP Layers

LayerFunction
Application Provides services like email, file transfer, web browsing.
Transport End-to-end communication, error control (TCP, UDP).
Internet Logical addressing and routing (IP).
Network Access Physical data transmission.

🔄 TCP/IP vs OSI Model

  • OSI model has 7 layers.
  • TCP/IP model has 4 layers.
  • OSI is theoretical.
  • TCP/IP is practical and widely used.

📞 Modem

  • Modem stands for Modulator and Demodulator.
  • Converts digital signals into analog signals.
  • Converts received analog signals back to digital.
  • Used for data transmission over telephone lines.
