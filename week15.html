Week 5 – Week 15 Notes

What is Internet?

The Internet is a global network of interconnected computers and devices that communicate using standardized communication protocols. It allows users to share information, access content, and interact with online services from anywhere.

History of Internet

  • 1960s – ARPANET: First network funded by the U.S. Department of Defense
  • 1970s: Email developed by Ray Tomlinson
  • 1980s: TCP/IP established as standard networking protocol
  • 1990: Tim Berners-Lee invented the World Wide Web
  • 2000s: Growth of broadband, social media, e-commerce, and mobile internet

Early Days of the Internet

  • Used only by research institutions and government agencies
  • Main purpose was communication and file sharing
  • No graphical interface – text-based commands were used

Evolution of the Internet

  • Shift from dial-up to broadband connections
  • Mobile internet evolution from 3G → 4G → 5G
  • Fiber optics for ultra-fast speeds
  • Satellite internet for remote areas (e.g. Starlink)

Web Link (URL)

A web link is a unique address used to access a resource on the Internet. It is also called a URL (Uniform Resource Locator).

  • Example: https://www.example.com
  • Typed in the browser address bar

Types of Web Links

  • Internal Links: Connect pages within the same website
  • External Links: Redirect to other websites
  • Hyperlinks: Clickable text, image, or button

Importance of Web Links

  • Website navigation
  • SEO and search engine ranking
  • Resource sharing and referencing

Browser

A browser is software that allows users to access and view content on the Internet.

  • Chrome
  • Firefox
  • Edge
  • Safari

Features of Browsers

  • Tabbed browsing
  • Bookmarking
  • Extensions / Add-ons
  • Privacy / Incognito mode

Internet Service Providers (ISPs)

ISPs provide Internet access and maintain the infrastructure required for connectivity.

  • PTCL
  • Comcast
  • AT&T

Types of Internet Connections

  • DSL
  • Cable
  • Fiber

Comparison of Internet Connection Types

Feature DSL Cable Fiber
Speed Up to 100 Mbps Up to 1 Gbps Up to 10 Gbps
Availability Widely available Urban areas Limited regions
Technology Telephone lines Coaxial cables Fiber-optic cables
Best For Basic usage Streaming & gaming High-demand apps

Difference Between ISP and OSP

Aspect ISP OSP
Definition Provides Internet connectivity Provides services over Internet
Primary Function Network access & bandwidth Content & services
Examples PTCL, AT&T, Comcast Google, Netflix, Dropbox
Role Connects users to Internet Enhances user experience
